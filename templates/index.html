Ya 3ayni Standings

A website that displays standings for ESPN Fantasy Basketball leagues in line chart format. The code works for (public) leagues that use Rotisserie format. Under the hood, this code scrapes the standings table and updates the charts on a daily basis. The source code for this project is available at this GitHub repository.
